Pet image assets live here.

Expected filenames:

- `<egg-id>-egg.webp`
- `<egg-id>-adult.webp`

Examples:

- `windfire-egg.webp`
- `windfire-adult.webp`
- `zodiac_dragon-egg.webp`
- `zodiac_dragon-adult.webp`

The app will prefer these raster assets when present. If a file is missing, `PetAvatar.vue` falls back to the lightweight code-rendered placeholder.
