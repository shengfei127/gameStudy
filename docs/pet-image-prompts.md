# Pet Image Asset Brief

本项目的宠物图保存在 `src/static/pets/`，命名规则为：

- `<egg-id>-egg.webp`
- `<egg-id>-adult.webp`

应用会优先加载这些写实图片资产；如果缺失，`PetAvatar.vue` 会回退到轻量代码绘制占位图。

## 风格方向

- 半写实、高质感、移动端收藏宠物图鉴风格。
- 每个角色都要有明显不同的性格、轮廓、材质和情绪温度。
- 该帅的帅、该温柔的温柔、该酷的酷，避免同一套卡通身体换颜色。
- 蛋形态需要能直观看出成年后的主题，例如颜色、纹样、材质、符号。
- 避免文字、水印、UI 边框、品牌 logo、直接复制已有影视/游戏角色。
